不论原版新版，在Ubuntu上都可以这样运行：
在命令提示符中，
1. 安装python：
sudo apt update
sudo apt install python3
2. 安装各类包：
pip install pygame
pip install cocos（实际上只用了cocos2d）
pip install pyglet
3. 解释python文件：
python3 gluttonous.py

然后用键盘、鼠标操作游戏。
